BOX MODEL

  • Box model represents how a content will be displayed on the web page.
  • The box model consist on 4 parts that are Content area , Padding , Border & Margin
  • The box model consists of the content area which can be anything like text or images which is surrounded by padding border and margin.

    • #1. Padding:-

  • Padding represents the space between the content and the border.
  • The padding size will always in px.
  • Padding with a single value will be applied on the four sides of the content.
  • Syntax :- padding: 10px;
  • Padding can be given on individual sides, which can be applied in the format in top right bottom and left.
  • Syntax :- padding-left: 30px; padding-top: 20px;;
  • We can also do like this instead of using padding left top keywords padding: 20px 40px 30px;.
  • In this method for first value it will start from top and rotate the values in clockwise manner.

    • #2. Border :-

  • Border for the content can be given using three values that are - size type color
  • Syntax :- border: size(px) type(solid,double,dashed,dotted) color;
  • Border radius property is used to curve the edges.
  • border-radius: 15px;
  • Border radius can also be given for individual edges.
  • Syntax :- border-top-right-radius: 35px;

    • #3. Margin:-

  • It define the space between the two diffrent elements in the web page.
  • We can also use margin to move an element in the web page.
  • It follows same syntax as padding.
