Position :

  • It is a special property of css with this we can set the positions of the elements in diffrents ways.
  • It contains five diffrent types of properties to set the postion of elements.
  • Five properties of position tags :
    1. static(default)
      • It is the default position of all the elements wherether we gave any type of postion property or not this will be there.

    2. Relative
      • It start from its original position means from its static position.
      • We can move a element by using these attributes :top,right,bottom,left .
      • Syntax : position: relative; top: 40px; left: 20px;

    3. Absolute
      • Its starting point is from top left point of web page.
      • We can move a element by using these attributes :top,right,bottom,left .
      • The element having higher z-index value will come on the top while over-lapping.
      • z-index value is not applicable for elements which having static position.
      • Syntax : position: absolute; top: 40px; left: 20px;

    4. Fixed
      • Its starting point is from top left point of web page.
      • It is working as absolute .
      • It will we fixed at the position what we have fixed.
      • We can move a element by using these attributes :top,right,bottom,left .
      • Syntax : position: fixed; top: 40px; left: 20px;

    5. sticky
      • It is as same as fixed.
      • In this top property is having more priority.
      • It will we fixed at the position what we have fixed.
      • We can move a element by using these attributes :top,right,bottom,left .
      • Syntax : position: sticky; top: 40px; left: 20px;

    It is as same as fixed.
