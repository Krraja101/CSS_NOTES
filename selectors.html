Selectors :

  • Selectors are used to put style internally or externally on the elements.
  • Basically we having 6 types of selectors to put styles :-
    1. Id selector (#)
      • Id means can be given to an element unique within a tag using id attribute.
      • The id names can be selected in css by using # Symbol.
      • Id has more priority then class selector.
      • Syntax :- #QSP{background-color: darkcyan;}

    2. Class slector(.)
      • Class names are not unique.
      • For class selector we use '.' for the styling.
      • If we want to keep the same style for more than one division we can use same class name for all the divisions.
      • Syntax :- .QSP{background-color: darkcyan;}

    3. Element selector(directly using div tag)
      • selecting the elements directly based on there names which will select all the respective element in the document.
      • Syntax :- h1{background-color: darkcyan;}

    4. Univesal selector(*)
      • * is used as a universal selector which selects all the element in the page.
      • Syntax :- *{margin:0;padding:0;}

    5. Desendent selector( .QSP h1{ background-color: red; } )
      • Selecting a particular child element within a div or a section.
      • Syntax :- .classnmae element { property : value ;}

    6. Group selector( h1,h2{color:chartreuse;})
      • Selecting the multiple element together in order to give common styles.
