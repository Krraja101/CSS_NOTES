Display Properties :

  • It represents how an html element will be displayed on the web page.
  • Every html element will have its own default display property.
  • There are five way to represent element in web pages :
    1. Block
      • Elements that occupy 100% width and displays one by one are called block level elements.
      • Ex:- heading,paragraph,br,hr

    2. Inline
      • Elements that displayed in the same row are called as inline elements.
      • Inline elements does not support height and width properties so we need to use inline-block property for merging the proeprty.
      • EX:- ancher tag.

    3. Inline-block
      • Elements which displayed in the same row and also supports height and width properties and called inline-block properties.
      • EX:-img,input tags,audio.

    4. Flex
      • This properties used to align a element in a row or colomn.
      • Flex should always be given for parent elements.

    5. None
      • This is used to remove the element from the web page.
      • Instead of this we can also use one property named visibility:hidden ; this property allow us to hide the particular element from the web page.
      • If we use visibility property to hide the particular element from the web page it will be at its position but not visible,that means it will not be removed.

    Some important properties of flex :

    # flex-direction :

  • It define the direction in which the flex items will be displayed.
  • All posibilities to decide the directions of flex is give below -

    • flex-direction: row-reverse;
    flex-direction: column;
    flex-direction: column-reverse;
    flex-direction: row;

    # justify-content :

  • This property is used to align the flex items on the x axis provided if there is sufficient amount of space in order to move the flex items.
  • All posibilities to arrange flex items in x-axis is give below -

    • justify-content: start;
    justify-content: center;
    justify-content: end;
    justify-content:space-between ;
    justify-content: space-evenly;
    justify-content: space-around;

    # align-items :

  • This property is used to align the flex items on the y-axis.
  • All posibilities to arrange flex items in y-axis is give below -

    • align-items: start;
    align-items: center;
    align-items: end;
