Pseudo Column & elements :

Pseudo Class :

  • It is user to add styles for the html only when the selectors meet the certain conditions.
  • It is defined using colon(:).
    1. active
      • It will work when ever we will click on the following element.
      • Syntax :- tag:active { background-color: goldenrod; color: aqua; font-size: 40px;}

    2. hover
      • It will work when ever we will move cursor on the following element.
      • Syntax :- tag:hover { background-color: goldenrod; color: aqua; font-size: 40px;}

    3. focus

    Pseudo element :

  • It is used to add styles by selecting a specific part of an element.
  • It is defined using double colon(::).
    1. first-line
      • This pseudo element put style for the first line of paragraph selected.
      • Syntax :- p::first-line{ background-color: blue; color: whitesmoke; font-size: 50px; }
