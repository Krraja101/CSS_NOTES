  • Basically CSS is classified in three parts:
    • Inline CSS
    • Internal CSS
    • External CSS

    1. Inline CSS

  • It is used for specific styling in the single tag.
  • We can achieve this by using attribute named as 'style' in any specific HTML tag
  • Inline css is having higher priority then all other types.
  • Style attribute contain lot of funtionality for html tags like (heading,paragraph)
  • Some example for INLINE CSS are :

    • Syntax :

    < h4 style="color: aqua;">CONTENT< /h4>

  • 'color' is a property of style attribute,it is used to put colours in the content.

    • INLINE CSS

    Syntax :

    < h4 style="color: blueviolet; background-color:coral;">CONTENT< /h4>

  • 'background-color' is a property of style attribute,it is used to put background for the given content.

    • INLINE CSS

    Syntax :

    < p style="background-color:pink;">CONTENT< /p>

    2. Internal CSS

  • It is used for putting style for a single html file.
  • We can achieve this by using 'style' as a tag inside head tag in HTML file.

    • Syntax :

    CASE I :- < style> h1{ color: royalblue; } p{ color: aqua; } < /style>

    CASE II :- < style> h1,p{ color: royalblue; } < /style>

  • This is used for putting one common style for a specific tag as(h1,p.....etc)
  • Where ever we will use tag which is mentioned inside the style tag it will having same style.
  • For CASE I we put seperate style for each tag (h1 & p).
  • But in CASE II we put same style for tags (h1 & p).

    • 3. External CSS

  • We can achieve this by using one seperate file with 'css' extention.
  • We can use this file for handling all kind of style related to html files and seperately we can link according to use.
  • By Using < link> tag we can connect CSS file with HTML file.
  • < link> tag contains href="" which refer to the css file.
  • < link> tag should be used within head tag.

    • Syntax :

    < link rel="stylesheet" href="index.css">

    NOTES :-

  • Inline CSS has higher priority then all other CSS.
  • If in some tags we put style using internal or external CSS and we also did some inline CSS there then the priority will be to the inline CSS.
  • But in case of internal and external CSS we dont have priority for them because it follows overridding property.

    • Syntax :

    CASE I :- < link rel="stylesheet" href="index.css"> < style> h1{ color: blueviolet; } < /style>

  • For CASE I the internal style will be executed because when the code will be executed line by line style defined by using internal CSS is after external CSS.

    • CASE II :- < style> h1{ color: blueviolet; } < /style> < link rel="stylesheet" href="index.css">

  • For CASE II the external CSS style will be executed because when the code will be executed line by line style defined by using external CSS is after internal CSS.
