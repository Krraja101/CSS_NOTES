Transform :

  • It is a CSS property is used to rotate, scale, skew, or move an element on web page.
  • It consists of some properties which helps us to do above operations.
    1. rotate
      • This property is used to rotate a particular element on web page.
      • It can rotate in both clockwise and anti-clockwise directions.
      • For rotating the element in the clockwise direction we need to pass +ve value in rotate attribute.
      • Syntax : transform: rotate(360deg);
      • For rotating the element in the anti-clockwise direction we need to pass -ve value in rotate attribute.
      • Syntax : transform: rotate(-360deg);

    2. scale
      • It is used to zoom in or zoom out an element on the web page.
      • For zoom in the element we need to pass +ve value to the scale attribute.
      • Syntax : transform: scale(5.0);
      • For zoom in the element we need to pass -ve value to the scale attribute.
      • Syntax : transform: scale(-5.0);

    3. translate
      • It is used to move an element on page in X or Y axis.
      • In this we having 3 attribute translate() , translateX() & translateY()

      • translate() is used to move an element in X-axis , if we will pass +ve value then it will move in right side of the page.
      • Syntax : transform: translate(200px);
      • If we will pass -ve value to translate() attribute then it will move to left side of web page.
      • Syntax : transform: translate(-200px);

      • translateX() is as same as translateX() both are used to move an element in X-axis , if we will pass +ve value then it will move in right side of the page.
      • Syntax : transform: translateX(200px);
      • If we will pass -ve value to translateX() attribute then it will move to left side of web page.
      • Syntax : transform: translateX(-200px);

      • translateY() is used to move an element in Y-axis , if we will pass +ve value then it will move in bottom of the page.
      • Syntax : transform: translateY(200px);
      • If we will pass -ve value to translateY() attribute then it will move to top of web page.
      • Syntax : transform: translateY(-200px);

    Some other attributes :

    1. transition :
      • It is used to control the speed of operations done by transforms property like rotate , scale or translate.
      • Lower the value higher the speed.
      • Syntax : transition:1s ;

    2. box-shadow :
      • It is used to put a shadow for the division ,image etc.....
      • It mainly contains four values X-axis,Y-axis,blur,colour
      • Syntax : box-shadow: 10px 10px 5px black ;

    3. box-shadow :
      • It is as same as box-shadow property.
      • It is used for putting shadow effect in texts.
      • It also mainly contains 4 values X-axis,Y-axis,blur,colour
      • Syntax : text-shadow: 10px 10px 5px black ;

    4. background-color :
      • It is used to put background color, for division,heading etc...
      • Syntax : background-color: blue;
    5. background-image :
      • It is used to put background-image for a page or a division.
      • For putting a image in background we need to use one attribute named url() in the background property.
      • Syntax : background-image: url(Image_link);
      • background-image: linear-gradient( rgba(0, 0, 0, 0.5), rgba(0, 0, 0, 0.5) ), url("rbjs.jpg"); background-size: contain; background-position: center; background-repeat: no-repeat;
    6. linear-gradient :
      • It is used to put gradient color for background.
      • It is always used with background properties like background or background-image.
      • When it is used with background then it will be used as fill the division with the gradient colors.
      • Syntax : background: linear-gradient(red,blue);
